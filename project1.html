ninteneleven

web design and development


"ninteneleven" is a make-believe video game company website designed and developed using HTML & CSS.

Roles: Designer & Developer

Timeline: Summer 2023, May - August

Team: Jasper Ling & one other peer

Project Design Process > > > > >

1. Setting up a Style Guide Template


We created a Style Guide Template to create a consistent design foundation for the website. All elements including buttons, links, colour, and content structure are detailed within the Style Guide Template.

2. Implementing the Style Guide


With our Style Guide in place, we developed the website's design following the templates we made. This enabled us to create a website design that was cohesive, and consistent across the site.
