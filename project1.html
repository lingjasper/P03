ninteneleven

web design and development


"ninteneleven" is a make-believe video game company website designed and developed using HTML & CSS.

Roles: Designer & Developer

Timeline: Summer 2023, May - August

Team: Jasper Ling & one other peer



Check out the website live!

Project Design Process > > > > >

1. Setting up a Style Guide Template


We created a Style Guide Template to create a consistent design foundation for the website. All elements including buttons, links, colour, and content structure are detailed within the Style Guide Template.

2. Implementing the Style Guide


With our Style Guide in place, we developed the website's design following the templates we made. This enabled us to create a website design that was cohesive, and consistent across the site.

To keep track of our progress, we used GitKraken to track our commits, and the changes we made.

3. Final Designs


Our project was finally completed after long collaboration periods, and a beautiful website was created. The website features 6 unique pages aside from the style guide, and each content card features its own subpage.

All images are sourced online, and credited.

Reflection


This project taught me how to collaborate on github projects, and communicating when to push and commit code when not in face-to-face settings.

The challenge of communication was a huge one, especially as my team had never used GitKraken before, nor coded through push and pull methods. So, attempting to push our changes and then having to resolve merging issues was a tough lesson to learn.
